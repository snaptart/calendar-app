üìö Sample Files & Format Guide

JSON Format Example

[
  {
    "title": "Team Meeting",
    "start": "2025-06-15 10:00:00",
    "end": "2025-06-15 11:00:00",
    "user_name": "John Doe",
    "description": "Weekly team sync"
  },
  {
    "title": "Project Review",
    "start": "2025-06-16 14:00:00",
    "end": "2025-06-16 15:30:00",
    "user_name": "Jane Smith"
  }
]

Required fields: title, start, user_name
Optional fields: end, description

CSV Format Example

title,start,end,user_name,description
Team Meeting,2025-06-15 10:00:00,2025-06-15 11:00:00,John Doe,Weekly team sync
Project Review,2025-06-16 14:00:00,2025-06-16 15:30:00,Jane Smith,
Client Call,2025-06-17 09:00:00,2025-06-17 10:00:00,John Doe,Important client discussion

Header row required. Leave empty cells for optional fields.

iCalendar (.ics) Format Example

BEGIN:VCALENDAR
VERSION:2.0
PRODID:-//Your App//Calendar//EN
BEGIN:VEVENT
UID:event1@example.com
DTSTART:20250615T100000Z
DTEND:20250615T110000Z
SUMMARY:Team Meeting
DESCRIPTION:Weekly team sync
ORGANIZER:John Doe
END:VEVENT
END:VCALENDAR

ORGANIZER field is used to match existing users.
